EcoQuest : Rahasia Alam


EcoQuest : Rahasia Alam adalah sebuah game yang bertemakan edukasi. Di dalam game ini akan diajarkan sebuah materi IPA yaitu ekologi dan keanekaragaman hayati.
Di dalam game ini, pemain akan ditugaskan meneliti makhluk hidup yang ada di alam sekitar.
EcoQuest : Rahasia Alam ini adalah game persembahan tim BrainByte, salah satu tim game developer SMP Negeri 1 Cilimus

Math Jump


Math Jump adalah sebuah game yang bertemakan edukasi. Di dalam game ini akan diajarkan sebuah materi Matematika dasar seperti penjumlahan, perkalian, dan pembagian.
Di dalam game ini, pemain akan ditugaskan melewati berbagai tantangan parkour dan menjawab soal Matematika untuk melewati suatu level
Math Jump ini adalah game persembahan tim Open Source, salah satu tim game developer SMP Negeri 1 Cilimus
